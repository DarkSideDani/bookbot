# BookBot
----
Bookbot is a simple command-line program that reads text from a file and generates a report about the text.

It will count 2 main things, the number of words within the document and how many times a specific character from the alphaber was found via the `[.isalpha()](https://docs.python.org/3/library/stdtypes.html#str.isalpha)` method.
